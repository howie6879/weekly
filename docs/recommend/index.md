### [Apk.1-Installer](https://github.com/Ium-Lab/Apk.1-Installer)

- 获取安装权限（不需要读取存储权限）
- 在QQ或微信里点击文件 → 用其他应用打开 → Apk.1 安装器 → 软件安装程序
