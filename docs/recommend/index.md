- 在线查看服务器 CPU、内存、系统负载、磁盘使用率等指标
- 在线 ChatGPT 助理
- 一键申请和自动续签 Lets encrypt 证书

- 基于网页浏览器的高级命令行终端
- 支持深色模式
- 自适应网页设计
