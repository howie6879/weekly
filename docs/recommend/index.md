- 新版支持多jar(爬虫包)，增加直播节目预告 EGP。
- 编辑器 PC Web版 [CatVodTVJsonEditor](https://catvodtvofficial.github.io/CatVodTVJsonEditor/)
- 配置地址---输入站源---确定即可 URL站源规则
