### [wechatDataBackup](https://github.com/git-jiadong/wechatDataBackup)

PC微信聊天记录数据导出工具

基于- wails开发 + React前端，实现PC端微信聊天记录一键导出功能。
- 导出后数据可以做永久化保存，即使微信停止支持，聊天记录也可以随时查看。
- 前端界面尽量与微信界面保持一致，减少使用成本。
- 理论上支持所有Windows 32/64位微信版本。
